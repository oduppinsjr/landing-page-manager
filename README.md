# 📄 Landing Page Manager for WordPress

A custom WordPress plugin for creating and managing client-specific marketing landing pages for towing companies.

---

## 📌 Features

- Registers a custom post type `Landing Page`
- Dedicated admin menu for managing landing pages independently of regular site pages
- Landing pages can follow custom templates, layouts, or styling separate from the site’s main pages
- Supports custom fields for:
  - Client name
  - Keywords
  - Landing page URL
  - Publish date
- Filter landing pages by client or keyword in the admin dashboard
- Export landing page data to CSV for reporting or analytics

---

## 📂 Installation

1. Download the plugin folder.
2. Upload it to your WordPress site's `/wp-content/plugins/` directory.
3. Activate the plugin via the WordPress Plugins menu.

---

## ⚙️ Usage

1. Navigate to **Landing Pages** in the WordPress admin sidebar.
2. Add or edit landing pages using the custom fields provided.
3. Filter, search, and export landing page data as needed.

---

## 📌 Requirements

- WordPress 6.x+
- PHP 8.x+
